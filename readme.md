### Features

- **Multiple Trading Profiles** - Create up to 5 independent accounts with isolated P&L, history, and chart data
- **Real-time Depth of Market** - Visual DOM ladder with bid/ask volume bars
- **Live Equity Curve**        - Beautiful green/red line showing your account growth
- **Trade Markers**            - Entry/exit arrows on chart with LONG/SHORT labels
- **Persistent Everything**    - All data survives refresh — never lose your progress
- **Ultra-Fast Chart**         - Lightweight Charts powered by Lightweight Charts — smooth even with 10k+ candles
- **Dark Mode First**         - Built for traders — sleek, minimal, professional
- **Hotkeys Ready**           - Designed for speed (coming soon)

---

### Tech Stack

- **React** + **TypeScript**
- **Zustand** - ultra-light state management
- **Tailwind CSS** + **shadcn/ui** - beautiful, accessible components
- **Lightweight Charts** - fastest trading charts on the web
- **Vite** - blazing fast dev server
